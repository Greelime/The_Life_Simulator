Симуляция представляет собой визуализацию процесса смены поколений некоторых существ, а также распростренения заболевания среди них.
Симуляция размножения представляет собой программную реализацию клеточного автомата Конвея.
Симуляция распространения заболевания работает по следующему принципу: клетка заражается с вероятностью 1/2, если вокруг 2 заболевших существа, с вероятностью 1/4, если вокруг 1 зараженное существо. Если вокруг 3 зараженных существа, то данное заражается наверняка.
Если вы хотите посмотреть код, запустить программу из IDE, зайдите в папку "Python" (у вас должны быть установлены неокторые дополнительные библиотеки)
Если вы хотите просто запустить программу, запустите файл "main.exe" в папке "For Windows"

--------------------------

The simulation is a visualization of the process of generational change of some creatures, as well as the spread of the disease among them.
Reproduction simulation is a software implementation of the Conway cellular automaton.
The simulation of the spread of the disease works according to the following principle: a cell is infected with a probability of 1/2 if there are 2 sick creatures around, with a probability of 1/4 if there is 1 infected creature around. If there are 3 infected creatures around, then this one is charging for sure.
If you want to view the code, run the program from the IDE, go to the "Python" folder (you must have some additional libraries installed)
If you just want to run the program, run the file "main.exe " in the "For Windows" folder
